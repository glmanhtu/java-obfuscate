Setting DashO Home

Setting DashO Home

The ant-dasho.jar file contains only the tasks and related classes, not DashO itself. The tasks need to know the location where DashO is installed and there are several ways to specify its value. The methods are listed in their order of precedence.

The dashohome attribute

Each task has a dashohome attribute where you can specify the directory where DashO is installed. If this attribute is specified it takes precedence over all other methods. For example:

<watermark dashohome="/user/lazardo/dashox.y">

The dasho.home property

You can set a buildfile property called dasho.home that specifies DashO's install directory. If specified it takes precedence over any of the following methods. For example:

<project name="superzap" basedir=".">
    <property name="dasho.home" value="${user.home}/DashO x.y"/>
</project>

The property can also be set from the command line:

ant "-Ddasho.home=$HOME/DashO x.y"

Setting dasho.home this ways will override any property setting done in your buildfile. Note that the entire argument needed to be quoted because the value contains a space.

dasho.ant.runtime.properties file

The ant-dasho.jar file can contain the dasho.ant.runtime.properties file which provides the a setting for dasho.home. You can display the current setting in this file by going to the ANT_HOME directory and running:

C:\>java -jar lib\ant-dasho.jar
dasho.home=C:\Program Files (x86)\DashO x.y
dasho.fullVersion=x.y.z
dasho.version=x.y

If this value is not correct, please see the instructions in Installation.

The environment

The DASHO_HOME environment variable can be used to specify DashO's install directory.

$ export DASHO_HOME=/user/lazardo/dashox.y
$ echo $DASHO_HOME
/user/lazardo/dashox.y


C:\>set DASHO_HOME="C:\Program Files (x860\DashO x.y"
C:\>echo %DASHO_HOME%
C:\Program Files\DashO x.y

Default locations

If no location is set the task will fall back to default locations where DashO would be installed. These locations vary by operating system:

  • Microsoft Windows - C:\Program Files (x86)\PreEmptive Solutions\DashO x.y
  • Mac OS X - no default currently defined
  • Unix/Linux - no default currently defined

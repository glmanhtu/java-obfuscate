DashO Samples

This directory contains samples that demonstrate using DashO.

Most samples contain an Ant build.xml file that can be used to compile, run DashO and run the samples. You do not need to install DashO's ant tasks to be able to use these build.xml files.

Some samples have specific setup and configuration instructions.

Note: If you are running the samples on a Windows machine and installed to the Program Files directory, you may need to run the command prompt, DashO, and other tools with administrator privileges.

samples/AndroidPA

An Android Application which uses PreEmptive Analytics to send messages. See here for special instructions.

samples/applet

A calculator Applet. Use run.html to see the obfuscated applet, after running DashO.

samples/instrumentation

A sample of instrumentation using PreEmptive Analytics. Please contact PreEmptive Solutions for additional details.

samples/jsp-pa

A sample WAR file which illustrates the use of the PreEmptive Analytics taglib in JSPs. See here for more details.

samples/log4j

Use log4j to generate PreEmptive Analytics messages and exception reporting without instrumentation. See here for special instructions.

samples/multidir

Cross-directory obfuscation using the merge="false" output option.

samples/multijar

Cross-jar obfuscation using the merge="false" output option.

samples/pa-api

A simple example using the Java PA API.

samples/simpleapp

A hello world type application.

samples/simpleappjar

The simpleapp application using a Quick Jar project.

samples/SpringBean

An Spring Application showing how DashO deals with Spring Beans. See here for special instructions.

Before running the Shelf Life samples please see here for special instructions.

samples/shelflife/basic

Use Shelf Life to add an expiration date to a hello world application.

samples/shelflife/customaction

Use a custom expiration action to bring up dialogs when the application has expired.

samples/shelflife/relativestart

Expire an application a certain number of days from a dynamic start date.

samples/shelflife/tokensource

Read the expiration token from an external source, in this case from resources. See here for special instructions.

samples/shelflife/authorized-app

Using Shelf Life to add an authorization check and a free trial period to an existing application. See here for special instructions.

Building and Running

This sample uses Ant to build and run. Use the compile and run targets respectively.

The pa_appender.xml contains parameters that define where PreEmptive Analytics messages are sent. You will need to change the companyId and applicationId when sending data to a PreEmptive Analytics server.

You can read more about the parameters for Log4JAppender in the javadocs.
